
///////////////////////////////////////////////////////////////////////////////////////
//                                                                                   //
// Telporter system that connects the towers in a ring and each tower to the castle. //
//                                                                                   //
// - Can be disabled with @celestia_warps_off                                        //
// - Can be enabled with @celestia_warps_on                                          //
// - If it was disabled it will stay disabled after restart since                    //
//   its sets the variable $CELESTIA_WARPS_OFF to 1                                  //
//   and if it's 1 the npcs will automatically disable themselves on init            //
//                                                                                   //
// Warp layout:                                                                      //
//                                                                                   //
//          T8L|T8M|T8R---T1L|T1M|T1R                                                //
//           /    \           /    \                                                 //
//          /      \         /      \                                                //
// T7L|T7M|T7R     C8       C1     T2L|T2M|T2R                                       //
//  |    \_______C7           C2_______/    |                                        //
//  |     _______               _______     |                                        //
//  |    /       C6           C3       \    |                                        //
// T6L|T6M|T6R     C5       C4     T3L|T3M|T3R                                       //
//          \      /         \      /                                                //
//           \    /           \    /                                                 //
//          T5L|T5M|T5R---T4L|T4M|T4R                                                //
//                                                                                   //
// C = Castle, T = Tower                                                             //
// 1-8 = Position in castle/Number of tower                                          //
// L = left, M = middle, R = right                                                   //
//                                                                                   //
///////////////////////////////////////////////////////////////////////////////////////

//////////////////////
// Castle to towers //
//////////////////////
